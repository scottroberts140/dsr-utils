# dsr-utils

Utility functions and helpers for common data science tasks.

## Features

- **String utilities**: Text processing and manipulation functions
- **Type utilities**: Type checking and validation helpers
- **Formatting utilities**: Data formatting and display helpers

## Installation

```bash
pip install dsr-utils
```

## Usage

```python
from dsr_utils import formatting, strings, types

# Use utility functions
result = formatting.format_output(data)
```

## Requirements

- Python >= 3.9
- pandas
- numpy
